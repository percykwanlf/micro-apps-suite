# Micro Apps Suite

A collection of high-utility, hyper-niche web applications designed to solve specific professional and hobbyist pain points.

## Included Apps:

1. **Olfacta**: A professional lab bench for perfumers to calculate formula percentages and ensure IFRA chemical compliance.
2. **ShiftNomad**: A real-time intelligence network for travel nurses to filter hospital contracts based on safety and pay.
3. **SporeSpot**: A geo-triggered tracking tool for wild food foragers to monitor micro-climate conditions for optimal harvest.

## Technologies Used:
- **Frontend**: React, Vite, Tailwind CSS
- **Deployment**: Vercel
- **Persistence**: Browser LocalStorage

## Purpose:
These applications are designed to solve "high-friction" problems for micro-communities by replacing manual spreadsheets and chaotic social media feeds with streamlined, automated tools.
